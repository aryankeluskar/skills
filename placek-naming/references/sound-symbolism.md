# Sound Symbolism & Name-Construction Reference

Source: David Placek / Lexicon Branding research (40+ years, 253+ linguists employed, Stanford/MIT/Berkeley collaborations). Use this when generating candidates or writing rationales in [../SKILL.md](../SKILL.md).

## Letter assets

Each letter carries a sonic *and* semantic signal. Bias name construction toward letters whose signal matches the framework's desired feeling.

| Letter | Sonic feel | Semantic / association | Named examples |
|---|---|---|---|
| **V** | Most alive, vibrant sound in English. Cross-cultural. | Speed, life, power, innovation. | Corvette, Viagra, Vercel |
| **B** | One of the most *reliable* sounds. Grounded, trustworthy. | Stability, technology when paired with hard consonants. | Blackberry (double B), IBM, Bank |
| **Z** | Noisy, energetic, attention-grabbing. | Modernity, electricity, edge. | Azure, Zoom, Zara |
| **X** | Fast, crisp, sharp. | Innovation, frontier, tech (aircraft, computers). | Xeon, X-ray, Xbox |
| **S** | Noisy (sibilant), flowing. | Sound, smoothness, motion. | Sonos (palindrome + double S), Swiffer |
| **K / hard C** | Sharp, decisive, percussive. | Strength, definitiveness, speed. | Kodak, Coke, CapCut |
| **P** | Plosive, punchy. | Crispness, premium when paired with -ium/-ial. | Pentium, Powerbook, Pepsi |
| **L** | Liquid, soft, flowing. | Calm, fluid, gentle, premium. | Lexus, Lululemon |
| **M** | Hummed, warm, maternal. | Comfort, smoothness, mass. | Mmm, Milk, Mazda |
| **N** | Nasal, neutral. | Negation-adjacent; use with care. | Nike, Nokia |
| **F** | Fricative, airy. | Lightness, freshness. | Fanta, Febreze |
| **R** | Rolling, strong. | Power when at start; warmth in middle. | Rolex, Ferrari |
| **D** | Reliable, blunt. | Solidity, decision. | Dell, Dyson |
| **T** | Crisp, terminal. | Precision, ending strength. | Tesla, Twitter |
| **G** (hard) | Guttural, weighty. | Power, weight, sometimes harsh. | Google, GoPro |
| **W** | Soft, open. | Wonder, openness, water. | Windsurf, Whisper |

## Construction rules of thumb

1. **Balance noise and flow.** Azure works because A (soft open) → Z (noise spike) → URE (smooth flow). Pure noise feels aggressive; pure flow feels forgettable. Aim for at least one signal letter against a smoother backdrop.

2. **Start strong.** First letter does disproportionate work. Hard consonants (V, B, K, P, X, Z) brand harder than soft openers (S, L, M, N).

3. **Familiar morphemes = processing fluency.** The brain is lazy. Vercel = "ver" (veritas, verde, very) + "cel" (accelerate, cell). Both halves are pre-loaded with meaning even though the whole word is coined. Always ask: *is each syllable already legible to the reader's brain?*

4. **Palindromes / symmetry are sticky.** Sonos reads the same forward, backward, and (visually) upside down. Symmetric names are memorable and visually stable as logos.

5. **Compounds are multipliers, not liabilities.** Lexicon's research: two familiar words combined produce more associative power than either alone. Powerbook = power + book → portable knowledge. Windsurf = wind + surf → flow + dynamism. Don't dismiss for length — Facebook is 8 letters and won.

## Direction buckets (use when generating)

When generating candidates, label each with one of these buckets so the user can see the spread:

- **Coined / morpheme-built** — invented from familiar parts (Vercel, Pentium, Sonos)
- **Compound** — two real words fused (Windsurf, Blackberry, Powerbook, Facebook)
- **Metaphor word** — real word that evokes the experience (Azure, Cursor, Swiffer, Stripe)
- **Mythological / classical** — Greek/Latin/Norse (Anthropic, Atlas, Anduril). **Warning:** this is the engineer default. Use ≤20% of the candidate list or you'll converge on the AI-startup cliché.
- **Cross-domain / "magazine trick"** — a word stolen from a totally unrelated field (sailing, cooking, architecture, hunting). These are where the *most surprising winners* come from per Placek.

## Tests a candidate must pass

Use this checklist when writing per-candidate rationales:

- [ ] **Pronounceable on first read** (processing fluency)
- [ ] **Spellable after hearing it spoken** (call-and-spell test)
- [ ] **Distinctive vs. the landscape** (doesn't echo competitor vocabulary)
- [ ] **Starts a story** — evokes an image or experience, doesn't describe a feature
- [ ] **Carries at least one sonic asset** from the table above
- [ ] **Survives 15 degrees of pivot** (won't break if the product evolves)
- [ ] **Could fire imagination** in a stranger ("they're not like the other guys")

## Anti-patterns to flag

- **Descriptive + category word.** CloudPro, CodeFlow, DataSync, AIAssist. These disappear.
- **Generic Greco-Roman.** Anthos, Kairos, Logos. Overused in 2024–2026 AI naming.
- **Compound with two weak halves.** Compounds only work when *both* halves carry meaning. "Datacore" loses; "Blackberry" wins.
- **Vowel-soup coined words.** Aiyoo, Eeola, Uaira. Hard to spell, no morpheme anchors, no fluency.
- **Numbers or punctuation as primary identity.** 8x8, 37signals, Co:llective. Workable but fragile — only when there's a real story.
- **Anything that sounds like an existing famous brand.** Even a single shared morpheme triggers "imitator" perception. Imitation is suicide (P&G president, ~50 years ago).
